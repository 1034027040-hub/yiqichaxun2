# 仪器查询与出入库管理系统

一个基于HTML、CSS和JavaScript开发的仪器查询与出入库管理系统，支持仪器信息管理、出入库登记、使用记录管理等功能。

## 功能特性

### 1. 仪器查询
- 支持按仪器管理编号进行搜索
- 仪器信息列表展示，包含编号、名称、规格、出厂编号、检定日期等
- 分页浏览功能，可调整每页显示数量

### 2. 仪器出入库管理
- 支持仪器出库操作
- 支持仪器入库操作
- 生成出入库台账，可打印和导出

### 3. 仪器使用记录
- 支持特定仪器的使用情况记录
- 可查看和编辑使用记录

### 4. 数据管理
- 本地存储数据，支持数据导入导出
- 自动保存功能
- 管理员模式，支持添加、删除仪器

### 5. 有效期管理
- 显示仪器检定有效期
- 支持更新仪器有效期

## 技术栈

- **HTML5**：页面结构
- **CSS3**：样式设计
- **JavaScript (ES6+)**：交互逻辑
- **LocalStorage**：数据存储

## 使用方法

1. **直接打开**
   - 下载项目文件
   - 直接在浏览器中打开 `index.html` 文件

2. **本地服务器运行**
   - 使用本地HTTP服务器（如Python、Node.js等）运行项目
   - 例如：`python -m http.server 8000`
   - 在浏览器中访问 `http://localhost:8000`
